ztf

Agent-assisted end-to-end testing for CLIs. Author scenarios in TOML, run them with one binary, get a single CI exit code — with an optional AI verdict for criteria too nuanced for a regex.

Install See an example GitHub
cargo install ztf

Why ztf

End-to-end tests where the success criterion is too nuanced for a regex — but you still want a single exit code for CI.

TOML-native scenarios

Define arrange / act / assert stages without writing test code. Each scenario is a self-contained, diff-friendly file.

Agent-in-the-loop verdicts

Delegate fuzzy checks ("does this output read like a polite greeting?") to a zag-driven AI reviewer that returns a structured JSON verdict.

Structured machine output

--format=json emits a stable envelope with file, scenario, assertion, and agent results — pipe it into jq or your CI.

Per-file temp dirs

Every command runs in a fresh temp directory exposed as $ZTF_TMP. Relative paths Just Work; cleanup is automatic.

Programmatic gates first

The agent only runs when stdout, stderr, exit code, and file checks all pass — saving tokens and keeping flakiness out of the agent path.

One CI exit code

0 if every scenario passed, 1 otherwise. Drops into any CI pipeline without bespoke scripting.

A scenario, end-to-end

The full examples/greet demo: arrange a tiny shell script, run it, assert against stdout and the filesystem, and let the agent decide whether the greeting reads naturally.

examples/greet/ztf.toml
$ ztf run examples/greet/ztf.toml examples/greet/ztf.toml ✓ greets a user by name assert exit_code = pass assert stdout_contains = pass assert file_exists = pass agent verdict = pass "Greets the user by name in a natural, polite way." ─────────────────────────────────────── summary total: 1 passed: 1 failed: 0 exit 0

Runs anywhere Rust runs

ztf is a single static binary with no runtime dependencies beyond an optional zag-supported provider on PATH for agent_review blocks.

Linux (x86_64, aarch64) macOS (Intel, Apple Silicon) Windows (x86_64) Rust 1.88+ (edition 2024)

Install

Pick whichever route fits your setup. ztf ships as a single binary; the zag-agent dependency is pulled in at build time.

via cargo 
cargo install ztf
from source 
git clone https://github.com/niclaslindstedt/ztf
cd ztf
make install

Need to run without an LLM provider? Set ZTF_SKIP_AGENT_REVIEW=1 to bypass agent_review blocks; programmatic assertions still run.
