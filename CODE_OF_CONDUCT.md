# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc-v2.1] as
its code of conduct. The full text is maintained upstream and applies
verbatim — please read it there:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

It applies to all project spaces: the repository, issues, pull
requests, discussions, and any associated chat channels.

## Reporting violations

To report a code of conduct violation, use the private reporting
channel described in [`SECURITY.md`](SECURITY.md). `SECURITY.md` is the
single source of truth for how to reach the maintainers privately —
this file intentionally does not duplicate that contact path so there
is only one place to keep up to date.

[cc-v2.1]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
